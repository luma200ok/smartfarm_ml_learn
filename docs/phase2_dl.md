## 1. 문제 (Problem)
- **ML이 못 하는 일**을 추가한다 → ① 잎 사진으로 병 진단(이미지), ② 환경 흐름 예측(순서).
- 특히 **잎 병해충은 정형 ML로 불가능 → CNN 필수** (이 프로젝트 DL의 핵심 명분).

## 2. 데이터 (Data)
- **비전:** PlantVillage (Kaggle, 즉시) / AI Hub 농작물 병해충(한국)
- **시계열:** 스마트팜코리아 / 공공데이터 / 가상 시뮬레이션
- _(클래스 분포·이미지 수 채우기)_

## 3. 방법 (Method)
- **CNN:** 전이학습 — **여러 백본 비교**(ResNet/EfficientNet) 후 베스트 선정 → 질병 분류
- **Grad-CAM:** 진단 근거 시각화(잎의 어느 병반을 보고 판단했나) — 설명가능 AI
- **LSTM/GRU:** 지난 N시간 환경 시퀀스 → 향후 추세 (+예측오차 임계값 이상탐지)
- **(선택) YOLO:** 잎 병반 위치 검출(바운딩박스) — 시간 여유 시
- 학습 안정화: 클래스 가중치(불균형)·학습률 스케줄러·EarlyStopping
- 평가: 정확도·혼동행렬·**ROC/AUC·오분류 분석**(비전), MAE(시계열), 학습곡선
